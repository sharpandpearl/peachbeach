## Easy Ways to Create the Image

### Option 1: Canva (Easiest - Free)
1. Go to https://www.canva.com
2. Create a design with custom dimensions: 1200 x 630 px
3. Set background to blue gradient (`#1e40af` to `#2563eb`)
4. Add text: "Mariliam" and "Clinical Research Consulting"
5. Download as PNG
6. Save as `og-image.png` in your `images/` folder

### Option 2: Figma (Free)
1. Go to https://www.figma.com
2. Create a frame: 1200 x 630 px
3. Add rectangle with gradient background
4. Add text layers
5. Export as PNG

### Option 3: Use an Online OG Image Generator
- https://ogimage.xyz/ (simple text-based)
- https://www.opengraph.xyz/ (quick templates)
